# e_commerce_data_analysis
Projet_stage_AnalystLab Afica
## Description
Ce projet présente une analyse complète d'un jeu de données e-commerce (Online Retail). L'objectif est de nettoyer les données, explorer les ventes, analyser le comportement des clients et produire des recommandations basées sur les données.

## Objectifs
- Nettoyer et préparer les données.
- Réaliser une analyse exploratoire des données (EDA).
- Identifier les produits les plus vendus.
- Analyser les performances des ventes par pays et par période.
- Identifier les meilleurs clients.
- Produire des visualisations et des recommandations.

## Outils utilisés
- Python (Pandas, NumPy, Matplotlib)
- Jupyter Notebook

## Résultats
Cette analyse met en évidence les tendances des ventes, les produits les plus performants, les principaux marchés, ainsi que des indicateurs permettant d'améliorer la prise de décision dans un contexte e-commerce.
